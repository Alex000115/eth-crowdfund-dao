# ETH Crowdfund DAO

A professional, flat-structure Solidity project implementing a decentralized crowdfunding platform. This repository demonstrates expert-level smart contract patterns, secure fund management, and campaign lifecycle logic on the Ethereum blockchain.

## Features

- **Campaign Creation:** Users can start fundraising campaigns with specific targets and deadlines.
- **Pledging:** Contributors can securely pledge ETH to campaigns.
- **Refund Logic:** Automatic refunds if a campaign fails to meet its target.
- **Owner Withdrawal:** Campaign creators can only withdraw funds if the goal is met within the deadline.
- **Flat Structure:** All files are located in the root directory for simplicity.

## Prerequisites

- Node.js (v14 or later)
- NPM or Yarn
